# Pilot Site Acquisition Priority

[VERIFIED: Josh Pappas call 2026-04-06]

At pre-seed stage, **pilot site acquisition** is the overriding priority. The ICP hypothesis is only as good as the pilots that validate it.

## Strategic Context

Kivira is pre-seed with investment from Wellstar's Catalyst arm. The company needs to quickly secure pilot sites to:

1. **Prove value** before scaling outbound GTM
2. **Build real ICP data** from actual customer conversations
3. **Generate patient behavior data** via text message before appointments
4. **Validate messaging hypotheses** with live feedback

## Implication for GTM

- Do not over-invest in TAM refinement before pilots validate the wedge
- Treat all messaging as hypothesis until pilot data confirms ([[MESSAGING_HYPOTHESIS_DISCIPLINE]])
- Prioritize speed-to-pilot over perfection in segmentation
- Use pilots to define the real ICP, not the other way around

## Call to Action

The immediate CTA in all outreach is **securing pilot sites**, not selling enterprise contracts. Frame every conversation around:

- Low-friction pilot setup
- Time-to-value for the practice
- Data collection that helps both parties
