# Revenue Accuracy Model (Risk Adjustment)

The Revenue Accuracy Model positions Kivira as a revenue-protection tool for Medicare Advantage plans and ACOs. The core thesis: PCPs routinely miss or under-code behavioral health diagnoses, leaving significant reimbursement on the table.

CMS uses Hierarchical Condition Categories (HCCs) to determine risk-adjusted payments. When a provider misses a diagnosis or under-codes severity (e.g., coding "Depression NOS" instead of "Major Depressive Disorder, Severe"), the plan loses per-member-per-month reimbursement.

## Key Points

- **Target buyer:** Medicare Advantage Plans, ACOs, risk-bearing provider groups
- **Value driver:** Closing the gap between actual patient acuity and coded diagnoses
- **Mechanism:** Kivira's clinical decision support identifies missed/under-coded BH conditions
- **V28 relevance:** CMS V28 model (fully phased in 2026) prioritizes severity accuracy over code volume

## The ROI Logic

The directional formula:

> (Total Members) x (Estimated % missed BH diagnoses) x (Avg. HCC value) = Recovered Revenue

The precise dollar values require verification against current CMS payment schedules, but the framework demonstrates that even modest improvements in diagnostic accuracy can yield significant revenue for risk-bearing entities.
